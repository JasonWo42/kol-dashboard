# KOL数据面板完整开发计划

## 目标

根据最新需求，实现一个完整的KOL数据面板，包含以下核心功能：

1. Campaign数据手动录入
2. KOL数据Excel上传自动化读取
3. D+7和D+30数据维度区分与展示
4. KOL视频明细详细信息展示

## 具体修改内容

### 1. 修改数据结构

#### Campaign数据结构

* 添加`totalBudget`字段

* 添加`dataD7`和`dataD30`字段，包含各自的播放量、互动量和分平台数据

* 添加`startDate`和`endDate`字段

#### KOL数据结构

* 在视频对象中添加`dataD7`和`dataD30`字段

* 添加`publishDate`字段

* 为每个视频添加`platform`和`platformIcon`字段

### 2. 实现Campaign管理功能

* 添加Campaign管理标签页

* 实现Campaign手动录入表单：

  * 产品名称输入框

  * Campaign名称输入框

  * 总预算输入框

  * 时间范围选择器

  * KOL数量输入框

  * 提交按钮

* 实现Campaign列表展示

* 实现Campaign编辑和删除功能

### 3. 实现KOL数据Excel上传功能

* 添加KOL管理标签页

* 实现Excel文件上传组件

* 添加文件上传预览功能

* 添加数据导入确认界面

### 4. 修改Campaign分析功能

* 添加D+7和D+30数据维度切换按钮

* 更新概览卡片，显示当前选中维度的数据

* 添加总CPV计算和展示

* 添加分平台数据展示区域，包含各平台的CPV

* 更新图表，支持不同维度的数据展示

### 5. 修改KOL表现功能

* 添加D+7和D+30数据维度切换按钮

* 在视频明细表格中添加：

  * 项目名称列

  * Campaign名称列

  * 视频发布时间列

  * 平台列（带图标）

  * 互动率列

* 更新表格数据，根据选中的维度显示对应数据

### 6. 界面优化

* 添加顶部导航栏，包含四个标签页：Dashboard、Campaign管理、KOL管理、设置

* 优化响应式布局，确保在不同屏幕尺寸下正常显示

* 添加数据加载状态和错误处理

* 优化表格排序和筛选功能

* 确保整体界面风格一致

## 技术实现

* 使用现有技术栈：React + Tailwind CSS + Chart.js

* 添加xlsx库用于Excel文件解析

* 实现文件上传和解析逻辑

* 保持代码结构清晰，易于维护

## 预期效果

* 完整的Campaign管理功能，支持手动录入和编辑

* KOL数据Excel上传功能，支持自动化读取

* D+7和D+30数据维度区分，支持切换查看

* 详细的KOL视频明细展示，包含完整的项目信息

* 美观的界面设计，良好的用户体验
